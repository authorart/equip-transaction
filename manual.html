2.2 ขั้นตอนการยืมอุปกรณ์
1
กดปุ่ม "ยืม/คืน/จองอุปกรณ์" ที่ Rich Menu
ระบบจะส่งข้อความ status และแสดง Flex Message รายการอุปกรณ์ทั้งหมด พร้อมสถานะว่าง/ไม่ว่าง
2
เลือกอุปกรณ์ที่ต้องการ → กดปุ่ม "📋 ยืม"
ปุ่มยืมจะแสดงเฉพาะอุปกรณ์ที่มีชิ้นว่างอยู่ ถ้าอุปกรณ์เต็มจะแสดงปุ่ม "⏳ จองคิว" แทน
3
กรอกแบบฟอร์มยืมอุปกรณ์
เบราว์เซอร์จะเปิดฟอร์มขึ้นมา กรอกข้อมูลให้ครบ:
ช่องรายละเอียดหมายเหตุ
เลือกชิ้นอุปกรณ์เลือก SubID ที่ว่าง เช่น CRRT-01จำเป็น
ประเมินสภาพอุปกรณ์สภาพก่อนยืม (Ready / Minor / Major)จำเป็น
ชื่อ-นามสกุลผู้ยืมจำเป็น
Ward / หน่วยงานเลือกจาก dropdownจำเป็น
เบอร์โทรติดต่อเบอร์ที่ติดต่อได้จำเป็น
วัตถุประสงค์เลือกจาก dropdownจำเป็น
วันที่คาดว่าจะคืนเลือกวันที่จำเป็น
หมายเหตุเพิ่มเติมข้อมูลที่ต้องการแจ้งไม่บังคับ
4
กด "📋 ยืนยันการยืม"
ระบบบันทึกข้อมูลและแจ้งเตือนเข้า LINE group อัตโนมัติ หน้าจอแสดงผลสำเร็จพร้อมสรุปรายการ
✓
เสร็จสิ้น — รับอุปกรณ์ได้เลย
กด "← กลับหน้าก่อนหน้า" หรือกดปุ่มย้อนกลับของ browser เพื่อปิดหน้าฟอร์ม
⚠️
หากประเมินสภาพอุปกรณ์เป็น "ชำรุดมาก/ใช้งานไม่ได้ (Major damage)" ระบบจะแสดงคำเตือน กรุณาระบุรายละเอียดในช่องหมายเหตุ และแจ้งผู้รับผิดชอบทราบด้วย
2.3 ขั้นตอนการคืนอุปกรณ์
1
กดปุ่ม "ยืม/คืน/จองอุปกรณ์" ที่ Rich Menu
ดู Flex Message แล้วหาอุปกรณ์ที่ต้องการคืน → กดปุ่ม "↩️ คืน"
2
เลือก SubID ที่ต้องการคืน
ระบบจะแสดงเฉพาะชิ้นที่ถูกยืมอยู่ เมื่อเลือก SubID แล้ว ชื่อ Ward และเบอร์โทรจะดึงข้อมูลอัตโนมัติจากการยืม
3
ประเมินสภาพอุปกรณ์ก่อนคืน
เลือกสภาพอุปกรณ์หลังใช้งาน และกรอกหมายเหตุ (ถ้ามี)
✅ สภาพพร้อมใช้งาน (Ready for use)
⚡ ชำรุดเล็กน้อย (Minor damage)
🚨 ชำรุดมาก/ใช้งานไม่ได้ (Major damage/Unusable)
4
กด "↩️ ยืนยันการคืน"
ระบบอัปเดตสถานะอุปกรณ์เป็นว่างทันที และส่งแจ้งเตือนเข้า LINE group พร้อมแจ้งคิวลำดับถัดไปอัตโนมัติ (ถ้ามีคิวรออยู่)

บทที่ 3
ระบบจองคิวอุปกรณ์
💡
ระบบจองคิวใช้เมื่ออุปกรณ์ที่ต้องการถูกยืมทั้งหมดแล้ว — ระบบจะแจ้งเตือน LINE ส่วนตัวทันทีที่อุปกรณ์ว่าง
3.1 วิธีจองคิว
1
ดู Flex Message → กดปุ่ม "⏳ จองคิว"
ปุ่มนี้จะแสดงเฉพาะเมื่ออุปกรณ์นั้นไม่มีชิ้นว่างเลย ถ้ายังมีว่างอยู่จะแสดงปุ่มยืมแทน
2
ระบบบันทึกคิวให้อัตโนมัติ
ไม่ต้องกรอกฟอร์มเพิ่ม ระบบใช้ชื่อบัญชี LINE ของคุณโดยอัตโนมัติ และแจ้งลำดับคิวที่ได้รับ
3
รอรับแจ้งเตือนทาง LINE
เมื่ออุปกรณ์ถูกคืน ระบบจะส่งข้อความส่วนตัว (DM) แจ้งทันที มีเวลา 2 ชั่วโมงในการมายืม
3.2 การแจ้งเตือนและการหมดอายุคิว
สถานการณ์สิ่งที่เกิดขึ้น
🔔 อุปกรณ์ว่าง LINE ส่ง DM แจ้งผู้จองคิวลำดับที่ 1
ข้อความ: "XXX ว่างแล้ว X ชิ้น! กรุณายืมภายใน 2 ชั่วโมง"
⏰ หมดอายุ (2 ชม.) คิวถูกยกเลิกอัตโนมัติ + DM แจ้งผู้ที่หมดคิว + ระบบแจ้งคิวลำดับถัดไป
🔄 คิวถัดไป ระบบแจ้งคิวลำดับที่ 2 โดยอัตโนมัติ (ถ้ามี)
🚨 เกินกำหนดคืน LINE group ได้รับแจ้งเตือนทุกเช้า 08:00 น. รายชิ้นที่เกินกำหนด
⚠️
หากได้รับแจ้งเตือนว่าถึงคิวแล้วแต่ยังไม่ต้องการยืม กรุณาปล่อยให้คิวหมดอายุ (2 ชั่วโมง) ระบบจะแจ้งคิวถัดไปอัตโนมัติ — ไม่จำเป็นต้องยกเลิกเอง
🔬

บทที่ 6
คำถามที่พบบ่อย (FAQ)
Qทำไมกดปุ่มยืมแล้วไม่มีปุ่ม "ยืม" ให้กด?
อุปกรณ์นั้นไม่มีชิ้นว่างในขณะนี้ ปุ่มจะเปลี่ยนเป็น "⏳ จองคิว" กดเพื่อจองคิวได้เลยครับ ระบบจะแจ้งเตือนทาง LINE ส่วนตัวทันทีที่อุปกรณ์ว่าง
Qได้รับแจ้งว่าถึงคิวแต่ยืมไม่ทันภายใน 2 ชั่วโมง ต้องทำอย่างไร?
คิวจะหมดอายุอัตโนมัติ ระบบจะส่ง LINE แจ้งว่าคิวหมดเวลา จากนั้นกลับไปกดจองคิวใหม่อีกครั้งได้เลยครับ
Qต้องการจองห้องส่องกล้องแต่ขึ้นว่าไม่มีสิทธิ์?
ระบบใช้ LINE User ID ในการตรวจสิทธิ์ กรุณาติดต่อผู้ดูแลระบบเพื่อเพิ่มสิทธิ์ การเพิ่มสิทธิ์ใช้เวลาไม่เกิน 5 นาที ไม่ต้อง Deploy ระบบใหม่
Qสามารถจองห้องส่องกล้องล่วงหน้าได้ไกลแค่ไหน?
จองได้ล่วงหน้าสูงสุด 4 สัปดาห์ นับจากวันที่จอง โดยระบบจะแสดงเฉพาะวันพฤหัสและศุกร์ที่ว่างและไม่ใช่วันหยุดนักขัตฤกษ์
Qกดปุ่มปิดฟอร์มยืม/คืนแล้วปิดไม่ได้?
เนื่องจากข้อจำกัดของ browser กรุณากดปุ่ม ← ย้อนกลับ หรือ ✕ ของ browser เพื่อปิดหน้าฟอร์มครับ ข้อมูลถูกบันทึกเรียบร้อยแล้ว
Qอุปกรณ์ที่ยืมเกินกำหนดจะเกิดอะไรขึ้น?
ระบบจะส่งแจ้งเตือนเข้า LINE group ทุกเช้าเวลา 08:00 น. รายงานทุกชิ้นที่เกินกำหนดคืน พร้อมชื่อผู้ยืม Ward และจำนวนวันที่เกิน
